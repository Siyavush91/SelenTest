Для удобства запуска тестов добавлены опции для запуска через указанный параметром для pytest браузер и url.
  - по умолчанию браузер "Chrome" и url - "https://marcojakob.github.io/dart-dnd/basic/".

Для запуска тестов с параметрами по умолчанию нужно набрать pytest в директории с проектом.
Для запуска тестов с указаннием параметров нужно набрать pytest --browser=<Название браузера> --url=<Нужный url> -s в директории с проектом
